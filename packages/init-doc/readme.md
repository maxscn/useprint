## Getting started

To get started, open a new shell and run:

```sh
npx init-doc@latest
```

This will create a new folder called `useprint-starter` with a starter document.

## Options

Pass a directory name as the first argument:

```sh
npx init-doc@latest foo
```

To pin `@useprint/components` and `@useprint/cli` to a specific npm dist-tag (default `latest`):

```sh
npx init-doc@latest --tag next
```
