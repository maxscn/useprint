# Useprint Starter

[docs](https://useprint.dev/docs/getting-started/automatic-setup)

A small Useprint project for creating printable documents with React.

## Getting Started

First, install the dependencies:

```sh
npm install
# or
yarn
```

Then, run the development server:

```sh
npm run dev
# or
yarn dev
```

Open [localhost:3000](http://localhost:3000) with your browser to see the result.

Documents live in `documents/`. Add new `.tsx` files there and export a React component as the default export.

To export HTML for every document:

```sh
npm run export
```
