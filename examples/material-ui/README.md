# Material UI PDF

[docs](https://useprint.dev/docs)

This example shows how to use Material UI with Useprint by rendering the MUI subtree with an Emotion cache, extracting critical CSS, and placing the generated HTML inside a Useprint document.

Install dependencies from the repository root:

```bash
bun install
```

Run the preview server from this example:

```bash
cd examples/material-ui
bun run dev
```

Export the document:

```bash
cd examples/material-ui
bun run export
```
